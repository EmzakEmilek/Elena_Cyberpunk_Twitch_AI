## 📋 Formát RAG kariet

Každá karta by mala obsahovať:
1. Základné informácie
2. História
3. Aktuálny stav
4. Vzťahy s inými entitami
5. Významné udalosti
6. Technické/špecifické detaily
7. Citáty a referencie
8. Meta informácie (zdroje, verzie)
